
## C-Lack
C-lack is a clone based of off Slack.com. It is essentially a chat room, designed to replace email.
<a href="https://c-lack.herokuapp.com">Live Site</a> 

### Impelmented Technologies:

- HTML
- CSS
- SASS
- JavaScript
- Postgres
- Ruby on Rails
- Webpack
- React
- Redux
